# One Man Fortress

## 使用说明

* WASD或上下左右控制角色运动

* 鼠标移动控制瞄准方向

* 鼠标左键或空格射击

* Q键循环切换武器，数字键直接选择武器

* R键重新开始游戏

* P键暂停游戏
