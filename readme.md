# One Man Fortress

## 内容

嵌入式系统作业，基于 Pygame 的平面射击打僵尸游戏

## 依赖环境

在 Python 3.9 环境下可运行

需要安装 Pygame 与 PyYAML 模块

## 使用说明

* WASD 或上下左右控制角色运动

* 鼠标移动控制瞄准方向

* 鼠标左键或空格射击

* Q 键循环切换武器，数字键直接选择武器

* R 键重新开始游戏

* P 键暂停游戏
